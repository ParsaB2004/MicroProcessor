کد ارائه شده، یک برنامه ساده برای کنترل دو LED بر روی برد آردوینو است. این برنامه باعث می‌شود LEDها به صورت متناوب روشن و خاموش شوند. در ادامه، هر خط از کد به طور جداگانه تحلیل خواهد شد.
int led1 = 13;
int led2 = 7;
سطر اول و دوم: 
دو متغیر با نام‌های led1 و led2 از نوع عدد صحیح (int) تعریف می‌شوند.
این متغیرها برای ذخیره شماره پین‌های دیجیتال آردوینو که LEDها به آن‌ها متصل شده‌اند، استفاده می‌شوند. در این مثال، LED اول به پین ۱۳ و LED دوم به پین ۷ متصل است.
void setup() {
  pinMode(led1, OUTPUT);
  pinMode(led2, OUTPUT);
}
سطر چهارم و پنجم: 
تابع setup() یک تابع خاص در محیط برنامه‌نویسی آردوینو است که یک بار در ابتدای اجرای برنامه فراخوانی می‌شود.
* pinMode(led1, OUTPUT); 
و pinMode(led2, OUTPUT);: این دستورات پین‌های led1 و led2 را به عنوان خروجی تنظیم می‌کنند. به این معنی که آردوینو می‌تواند به این پین‌ها ولتاژ اعمال کند و LEDها را روشن یا خاموش کند.

void loop() {

  digitalWrite(led2, LOW);

  digitalWrite(led1, LOW);
  digitalWrite(led2, HIGH);
  delay(1000);
}
سطر هفتم: 
تابع loop() یک حلقه بی‌نهایت است که به طور مداوم تکرار می‌شود. تمام دستوراتی که درون این تابع قرار دارند، به صورت پی در پی اجرا می‌شوند.
سطر هشتم تا دوازدهم: 
* digitalWrite(led1, HIGH);: 
این دستور باعث می‌شود که LED متصل به پین led1 (یعنی پین ۱۳) روشن شود. با ارسال مقدار HIGH به یک پین خروجی، به آن پین ولتاژ ۵ ولت اعمال می‌شود.
* digitalWrite(led2, LOW);: 
این دستور باعث می‌شود که LED متصل به پین led2 (یعنی پین ۷) خاموش شود. با ارسال مقدار LOW به یک پین خروجی، به آن پین ولتاژ ۰ ولت اعمال می‌شود.
* delay(1000);:
 این دستور باعث می‌شود که برنامه به مدت ۱۰۰۰ میلی‌ثانیه (۱ ثانیه) مکث کند.
سطرهای بعدی نیز به همین ترتیب عمل می‌کنند و باعث می‌شوند LEDها به صورت متناوب روشن و خاموش شوند.

Const int led = 8;

void setup() {
  pinMode(led, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  if (Serial.available() > 0) {
    char inputLed = Serial.read();
    if (inputLed == 'h' || inputLed == 'H') {
      digitalWrite(led, HIGH);
    } else if (inputLed == 'l' || inputLed == 'L') {
      digitalWrite(led, LOW);
    }
  }
}

#include <math.h>
const int numPoints = 360;
const float amplitude = 100;
const float offset = 512;

void setup() {
  Serial.begin(9600);
}

void loop() {
  for (int i = 0; i < numPoints; i++) {
    float radians = i * (PI / 180);
    float sineValue = amplitude * sin(radians) + offset;
    float cosValue = amplitude * cos(radians) + offset;
    Serial.print(sineValue);
    Serial.print(",");
    Serial.println(cosValue);
    delay(10);
    }
}
